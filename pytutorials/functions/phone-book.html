While list-like data structures can be fast and efficient, sometimes we want to be able to refer to things with descriptive names instead of just numbers. In this tutorial, we'll be applying one of Python's more versatile data types--the dictionary--practically.

Prerequisites

Before attempting this tutorial, you should of an idea of how to:
  • access and store information in dictionaries
  • create custom functions
  • use the for loop

Hopefully you are comfortable with the material listed above, because this tutorial will generally gloss over those subjects. If you find yourself getting stuck with anything on the above list during this tutorial, then try reading the relevant material from previous chapters before hopping back in.

Goal

Let's suppose we want to create a phone book in Python that looks something like this: 
phone_book = {
    'George Washington': 8057041776,
    'Thomas Jefferson': 5107041776,
    'John Hancock': 4157041776
    'Paul Revere': 5104181775
}

Furthermore, we want to make this phone book useful. That is, we want to implement functions that allow us to say, for example, search through it by name and number.

Step 1: Create a function to insert numbers

This function should have three arguments:

  • book - A dictionary containing the phone book
  • name - The person's name
  • number - The person's phone number

Solution to Step 1

def book_insert(book,name,number):
    phone_book[name] = number

Step 2: Create a function to print out our phone book

Hint: Remember that you can iterate over the keys of a dictionary. The following code

fruits = {
    apple: "A red fruit."
    orange: "An orange fruit."
} 

for i in fruits.keys():
    print(i)

will produce (in a random order): 

orange
apple

Solution to Step 2

def book_print(book):
    for name in book.keys():
        number = book[name]
        print(name,number)

Step 3: Create a function to search through our phone book

Let's have this function search through our phone book, printing anything that matches our search parameters. For now, let's just have one search parameter: a number. This number can be a full phone number, or just a few digits, e.g. an area code.

Hint 1: Checking if X is in Y 
"Slim Shady" in "Will the real Slim Shady please stand up?" will return True.

Now, if you stored your phone numbers as integers (like I did), then using the above code will produce an error since the in clause only works on iterables. The easiest solution is to convert your numbers to strings with Python's built-in str() function, like:

str(100) in str(10075)
Hint 2: Don't repeat yourself

Earlier, we already made a function book_print() that takes a dictionary as input, and spits its contents back out. Use this function again to print your search results. In order to do this, you'll also need to store your search results in a dictionary.

Step 3 Solution

def book_search(book,number):
    matches = {}
    for name in book.keys():
        phone_number=int(book[name])
        if str(number) in str(phone_number):
            matches[name]=phone_number
    book_print(matches)
