Goal

The goal of this exercise is to create a program which prompts the user to enter a speed in miles per hour and then prints out the conversion to kilometers per hours.

An example usage of the program should look like this:

  • Print the conversion in a specific format
  • Print an error message for unexpected (non-numeric) inputs

Some Know-How

Dealing with Data Types

Python's input() function returns all user input as a string. If you recall, strings have a handy .isnumeric() method for checking if a string is only composed of numbers. But remember to be careful, as the * operator works differently on strings than it does on numbers. 

>>> "500"*5
'500500500500500'
>>> 500*5
2500
>>>

Using the * operator on strings causes them to be repeated, since Python doesn't interpret "500" as "the number 500". Fortunately, Python has a float() function to convert numerical strings back into decimal numbers.

>>> x = "500"
>>> x*5
'500500500500500'
>>> float(x)
500.0
>>> float(x)*5
2500.0
>>>

Program Structure

Since we want this program to keep on going forever until the user stops, we'll want to use a while loop. Secondly, since there are multiple cases this program should respond to, we would probably want to use an if/else if/else type structure.

Lastly, but not leastly, use the formula: mph = 1.60934 * km/h. When you're done, try entering the same inputs as I did above to see if you get the same results.

Solution 

while True:
    mph = input("Enter a number to convert to kilometers per hour: (or nothing to quit) \n")
    if str(mph).isnumeric():
        kmh=float(mph)*1.60934
        print(str(mph) + " MPH is equal to " + str(kmh) + " KM/H\n")
    elif not mph:
        break
    else:
        print("Please enter a valid input.\n")
