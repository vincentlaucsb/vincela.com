Now that we have a basic understanding of if/else statements, let's go over some of the useful ways we can apply them.

A Cool Trick with Boolean Statements

In Python, empty strings, lists, dicts, and other container objects evaluate to False.

>>> bool("Just gonna stand there and watch me burn")
True
>>> bool([1,2,3,4,5])
True
>>> bool({"staples":500,"pens":10,"notebooks":5,"post-its":200})
True
>>> bool("")
False
>>> bool([])
False
>>> bool({})
False

Why does this matter?

There are many times when we're coding where we will need to check if, for example, a list is empty or not. In Python, we have a neater, shorter way to check if something contains items or not.

Example

Instead of:

if len(my_list) not 0

Simply use:

if len(my_list)

Checking for Membership (of a list, dictionary, etc.)

We can easily check if x is an element if a collection of elements, such as a list, by using the in operator.

Example: Using membership tests in if/else statements

vegetables = ['artichoke','carrot','celery','pizza']
food = 'pizza'

if food in vegetables:
    print(food + ' is a vegetable')
else:
    print(food + ' is not a vegetable')

Output

pizza is a vegetable
