Find all of the square numbers from 0 to 1000

An integer is called a square number when its square root is also an integer. For example, 36 is a square number because its square root, 6, is also an integer.

Using a while loop to find square numbers, place your results into a list and print the final list.

Solution 
n = 0
squares = []

while n**2 < 1000:
    squares.append(n**2)
    n += 1

print(squares)

First, we initialize an empty list to store our numbers in. Then, we simply square the integers 0, 1, ... n until n squared exceeds 1000.

Find the first 10 square numbers

Using a slight modification to the solution from the previous problem, generate just the first 10 square numbers.

Solution 
n = 0
squares = []

while len(squares) < 10:
    squares.append(n**2)
    n += 1

print(squares)

Generate the same list, but with a for loop

Solution 
squares = []

for n in range(0,11):
    squares.append(n**2)

print(squares)

Remove every other letter from a word

Create a function which removes every other letter from a word, starting with the second letter.

Solution:

How do you prooooonounciate that?

Solution: 
def pronounciate(input):
    vowels = ['a','e','i','o','u']
    result = []
    for word in input.split(' '):
        for letter in word:
            if letter in vowels:
                result.append(word.replace(letter,letter*5,1))
                break
        
    return ' '.join(result)
