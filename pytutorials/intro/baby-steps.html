Installing Python

Installing Python is very simple, as long as you have administrator privilleges on your computer.

Downloading from www.python.org (Windows, Mac, Linux)

Python can be downloaded from https://www.python.org/. You may need to scroll down to see it, but the download link is located directly on the main page.

Now, there are currently two main versions of Python in circulation—Python 3 and Python 2. For the purposes of this tutorial, we will be using Python 3.

Download from the Terminal (Mac, Linux)

If you are unsure how to access the command line (terminal), a quick Google will give you your answer. Once you have opened your terminal, enter this command:

sudo apt-get install python3

Note: Using "sudo apt-get install python" will install Python 2, not 3.
